# Mobile Application for Patients

This project was developed as part of a bachelor’s thesis titled  
**“Mobile Application for Patients”**.  
The application was created in 2022 and has not been updated or maintained since then.

---

## Project Purpose

The application is designed for patients and provides features such as:
- medication reminders,
- managing medical appointments,
- recording symptoms,
- tracking health measurements (weight, temperature, blood pressure, blood sugar).

---

## User Features

### Patient
- adding, editing, and deleting medication information,
- managing medical appointments,
- recording symptoms,
- maintaining a health measurements diary.

---

## Technologies

- C# (.NET, Xamarin)
- SQLite
